Project: NutriSnap AI

Goal: Users upload food photos and receive nutritional insights.

Core Features:
- Authentication
- Food Upload 
- Nutrition Analysis
- Daily Diary
- Weekly Charts
- Streak Tracking

Tech Stack:
- React
- Tailwind CSS
- Node.js
- Express.js
- MongoDB

# User Journey
1. Users registers an account
2. Users logs in 
3. User lands on dashboard
4. User uploads a food image
5. System anaylzes food
6. Nutrition information is displayed
7. Enrty is saved to diary
8. Dashboard updates calories/macros
9. User views weekly progress
10. user maintains streaks
